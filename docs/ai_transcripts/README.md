# AI Development Transcripts

This project was built with AI assistance (Claude Code). The assignment requires
submitting the AI development transcripts; they belong in this folder.

## What to add here

Export the development session(s) and drop them in this directory, e.g.:

```
docs/ai_transcripts/
  session-2026-06-16-skeleton-and-agent.md
  session-2026-06-17-evals-and-polish.md
```

(Markdown or plain text — whatever your client exports.)

## Decision trail (already in-repo)

Until the raw transcripts are exported, the **distilled** record of how the
project was steered — every significant decision and course correction, with
rationale — lives in [../steering_log.md](../steering_log.md). The design
rationale and results are in [../rationale.md](../rationale.md). Together these
summarize what the transcripts contain; the raw transcripts add the verbatim
back-and-forth.
